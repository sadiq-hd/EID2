```
index.html          ← الصفحة الرئيسية

components.css      ← المكونات التفاعلية

```

---

## شرح الملفات

### `style.css`
هيكل الصفحة الأساسي. فيه:
- الخلفية بالتدرجات الخضراء والزرقاء
- موضع النص في وسط الشاشة بشكل responsive
- حجم الخطوط وتأثير الـ shimmer على النصوص
- الزخارف في الزوايا الأربع
- الـ breakpoints للشاشات المختلفة (جوال، تابلت، شاشات كبيرة)

### `components.css`
المكونات التفاعلية. فيه:
- الشريط السفلي بالستيكر والنص والعداد
- نافذة الرسم اللي تطلع لما تضغط على هيكساغون فاضي
- نافذة عرض الرسالة المحفوظة
- التوست اللي يظهر بعد إرسال المعايدة

### `floatCanvas.js`
الأشكال الطائرة في الخلفية فقط. يرسم:
- هيكساغونات خضراء صغيرة تطفو للأعلى
- أهلة ذهبية
- نجوم خماسية وسداسية
- كل شيء يتحرك بشكل عشوائي ويختفي تدريجياً
- مستقل تماماً ولا يتكلم مع بقية الملفات

### `hexGrid.js`
قلب الموقع. فيه كل المنطق الأساسي:
- يبني شبكة الهيكساغون على الجانبين (يمين ويسار)
- يحفظ رسائل المستخدمين في الـ `localStorage`
- يتحكم في الضغط والـ hover على الهيكساغونات
- يفتح نافذة الرسم على الهيكساغونات الفاضية
- يعرض الرسائل المحفوظة لما تضغط على هيكساغون ممتلئ
- يحدّث العداد تلقائياً

| الملف | الدور |

## ملاحظات تقنية

- الشبكة تستخدم **HiDPI rendering** — تتكيف مع شاشات Retina عبر `devicePixelRatio`
- الرسائل تُحفظ بـ **transparent PNG** بدون خلفية بيضاء
- الهيكساغونات الجانبية تبتعد عن منطقة النص تلقائياً عبر `DEAD_TOP_PCT` و `DEAD_BOTTOM_PCT`
- الموقع **fully responsive** — يشتغل على كل الشاشات
